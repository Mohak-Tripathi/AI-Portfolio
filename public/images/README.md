# Project Images

Place your project images in this folder.

## Guidelines

- **Format**: JPG, PNG, or WebP
- **Size**: Recommended 800x600px or 1200x800px
- **Naming**: Use descriptive names (e.g., `chatbot-project.jpg`)
- **Optimization**: Compress images to reduce file size

## Placeholder Image

Until you add your actual project images, create placeholder images or:

1. Take screenshots of your deployed Streamlit apps
2. Create project mockups
3. Use actual app interface screenshots

## Usage

Reference images in `src/data/projects.ts`:

```typescript
image: "/images/your-project-name.jpg"
```

Note: The path starts with `/images/` (relative to the public folder).
